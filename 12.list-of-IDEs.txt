Short list of popular IDEs.

1. Eclipse
 The interface is a lot cleaner and concise than Visual Studio and the languages that are possible to use extend past the .Net Framework. 
 Besides being the number on way to develop Android applications via the Android SDK, it is fully extensible to use languages such as Ruby,
 C, C++ and many many more.
 
2. Netbeans
his program is extensible and free, and it compiles code which is a bonus. 
When installing Netbeans it comes with a Java server called GlassFish that you could deploy on your computer for your testing environment. 

3. Kate
Kate is just a text editor. 
Not to be looked down upon as it does a lot more than a regular text edit can do and that is what makes this an indispensable product.

4. Gedit
Gedit is the official editor of the gnome desktop and comes installed once the system is setup. 
Very useful for the user that is casually editing code and not undertaking major projects. 
This would be a good companion to Netbeans or Eclipse.

5. Notepad++
Notepad++ is a text editor and source code editor for Windows. 
It aims to be a lightweight and robust editor for a variety of programming and scripting languages.
Notepad++ opens large files significantly faster and can be used as a replacement for Windows Notepad
