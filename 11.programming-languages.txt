Short list of popular programming languages.

1. C-Sharp 
C-Sharp is a programming language produced by Microsotf as a part of its .NET. 
It combines principles from C+ and C++ and it is mostly used for development of software for Microsoft.

2. Java
Java is a class based and project-orientated programming language. It is designed to pork across multiple platforms including android and windows.
Both C# and Java drew from the C/C++ (and Objective C, and others) to define their syntax. And both of them are compiled to an intermediate language.

3. Objective-C
Objective-C is a general-purpose, object-orientated programming language used by Apple OS.
It is mostly used for crating apps for iPhone.
Objective-C is 
An Objective-C class has both an interface and an implementation. These are usually stored in separate files 

4. PHP
PHP is a free, server-side scripting language designed for dynamic websites and app development.
It can be directly embedded into an HTML source document rather than an external file, which has made it a popular programming language for web developers.
PHP can instantiate and use .NET objects

5. Python
Python is a high level, server-side scripting language for websites and mobile apps. 
It's considered a fairly easy language for beginners due to its readability and compact syntax.
Some python features can be mapped to C#.
